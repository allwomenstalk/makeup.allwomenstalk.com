# Should You Actually Try Using Eyebrow Tattoos?

If you haven't heard by now, there's a new product on the market. They're eyebrow tattoos and they're exactly what they sound like. Basically, you place a temporary tattoo of an eyebrow on your face, so that you don't have to fill them in.
