# The Best Nude Lipstick for Every Skin Type ...

Wendy Rowe, makeup artist and beauty artistic consultant at Burberry, opened up to ELLE about the major trend that is [nude lips](https://makeup.allwomenstalk.com/tips-for-perfect-nude-lips/) and which ones look best on each skin type. As fun as it is to go for a moody look during the fall with some dark lipstick, nude lips are having a major moment. Check out which kind YOU should be wearing:

## 1. Fair

![Fair](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/tu/z1/ao6wtgbc_600x848.jpg)

For fair complexion, Wendy Rowe recommends a nude lipstick with pink undertones! Try the Tom Ford Beauty Lip Color in Nude Vanille, $50 ([neimanmarcus.com](http://www.neimanmarcus.com?ref=allwomenstalk.com)).

## 2. Olive

![Olive](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/tu/ug/finih5rh_500x456.jpg)

If you have olive skin, a nude lipstick with pink undertones works just as well for you. I would pick a matte texture because the color seems to last a lot longer! Try Revlon Matte Lipstick in Nude Attitude, $8; ([drugstore.com](https://www.walgreens.com/)).

## 3. Medium

_ For a medium to dark complexion, Wendy suggests a nude lipstick with an orange undertone to it. Try NARS Satin Lipstick in Bilbao, $26 from barneys.com_.

## 4. Dark

![Dark](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/r0/tc/mgw06mmh_600x880.jpg)

If you have a dark complexion, try this beautiful Chanel Rouge Allure Intense Long-Wear Lip Color in Farouche, $35, [chanel.com](http://www.chanel.com?ref=allwomenstalk.com).
